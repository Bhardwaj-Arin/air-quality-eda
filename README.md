# PM2.5 Air Quality — Statistical Analysis

## Overview
Statistical analysis of PM2.5 air quality data across 
Indian cities (2015–2020) using EDA, distribution 
fitting, and hypothesis testing.

## Key Highlights
- Distribution fitting (Log-Normal, Exponential)
- Seasonal trend analysis
- Hypothesis testing (Winter vs Summer PM2.5)
- City-wise pollution comparison
- Correlation analysis of pollutants

## Tools
Python | Pandas | NumPy | Matplotlib | Seaborn | Scipy

## Dataset
Air Quality Data in India — Kaggle (2015–2020)

## Results
- PM2.5 best fits Log-Normal distribution
- Winter PM2.5 significantly higher (p < 0.05)
- Delhi ranks highest in average PM2.5
